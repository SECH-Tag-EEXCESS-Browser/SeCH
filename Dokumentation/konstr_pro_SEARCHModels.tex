\section{SEARCHModels}
Das \lstinline|SearchModels| stellt die Schnittstelle zwischen der SearchExtraction und dem TaskController dar. Von der SearchExtraction erzeugt, beinhaltet ein SearchModels-Objekt eine Sammlung von SearchModel-Objekten erzeugt aus einer Website. In einem \lstinline|SearchModel|-Objekt werden Search-Tag-Links mit der zugeh√∂rigen Search-Tag-Section/-Head und Filtern gespeichert.\newline
Zur globalen Identifikation wird in einem \lstinline|SearchModel| die Url der Website von der es kommt gesetzt. Um ein \lstinline|SearchModel| innerhalb einer Seite zu finden wird der Index gesetzt.
